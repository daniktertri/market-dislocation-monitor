# Crypto News Monitor

A demonstration site for monitoring and structured analysis of cryptocurrency-relevant news.

## Features

- **Home**: System overview with flow diagram
- **Demo**: Real-time news feed table with mock data
- **Performance**: Equity curve and drawdown analysis
- **Methodology**: Detailed system documentation

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- No backend - all data is mocked

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Design Philosophy

The site uses an austere, terminal-inspired design:
- Monospace typography
- Strict rectangular layouts
- Minimal color palette (black, white, gray, one accent)
- Pixelated aesthetic
- No rounded corners
- Professional, credible appearance

## Deployment

Deploy to Vercel or any Next.js-compatible hosting platform.
